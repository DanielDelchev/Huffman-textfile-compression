Vertex
Objects of the Vertex class are used to build the Huffmann Tree in the Tree class. A Vertex contains data for the Vertex itself, pointer to left successor and pointer to right successor in constrast to the objects of the Node class which contain no pointers and therefore can be serialized easily.Objects of this class are used to build the frequency table, the Huffmann tree and the serializable Nodes used for the serialiszble NodeTree.This class is in the basis of the structure of the program.
