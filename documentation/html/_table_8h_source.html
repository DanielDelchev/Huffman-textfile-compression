Table::read_table
void read_table(const char *from)
Reads a Table from a file and assigns its info to the object that called the function. Fist the count (size) is read and assigned to the current object.Then count Vertex are read into a temporary Vertex which is then added to the table with the function add_vertex(const Vertex* given).Lastly the limit(capacity) is read and also assigned to the current object This function used to be used when the Node and NodeTree classes were not yet available and serializatio and desirialization of the Huffmann tree was not anticipated.Currently it is not in use.Yet it is completely functional.
