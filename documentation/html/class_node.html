Detailed Description

Objects of this class are similliar to the objects of the Vertex class. Nodes are constructed from Vertices,keeping their letter, and relations. The main difference is that Nodes do not need pointers to store the relations. ID-s (Key values) are used to store the relations between Nodes.This way objects from Node class can be serialized and deserialized , while objects of the Vertex class cannot.
